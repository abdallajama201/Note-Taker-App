[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description

This project is a website that can be used to write and save notes. This project can be useful to anyone that wants to keep organized and keep track of things.

## Usage

The website is live on the open web and can be accessed at https://damp-spire-10778.herokuapp.com/ 

A user can begin by clicking the get started button, which will take them to the notes page. A user can view notes by clicking them and add a note by pressing the plus icon. Once the note is finished a user can save it by clicking the save icon. Notes can be deleted by clicking the delete icon next to the note they want to get rid of.
